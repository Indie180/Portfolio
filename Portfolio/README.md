## Setup

1. Clone or download this repository
2. Place your resume PDF in the `CERtificate` folder as `Indiphile_Tini_Resume.pdf`
3. Update contact information in `index.html` (email, LinkedIn, GitHub links)
4. Open `index.html` in a web browser

## Customization

### Colors
Edit CSS custom properties in `styles.css` (lines 10-30) to change the color scheme.

### Content
Update your information in `index.html`:
- Projects (lines 90-200)
- Achievements (lines 210-280)
- Skills (lines 290-400)
- Experience (lines 410-480)
- Contact info (lines 490-520)

### Fonts
Change fonts by updating the Google Fonts import in `styles.css` (line 6).
